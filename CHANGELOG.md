# Changelog

Bu projedeki tüm öenmli teknik güncellemeler aşağıda yer almaktadır

## 11 Mart 2026

### Eklenenler
- Proje altyapısı Vite + React ile oluşturuldu.
- BEM standartlarına uygun `Button` bileşeni eklendi.
- CSS Değişkenleri ile renk yönetimi altyapısı kuruldu.
- Erişilebilirlik odaklı `Input` bileşeni ve sayfa düzeni eklendi.
- Dark/Light mod eklendi.
- Mobil (≤640px), Tablet (641px-1024px) ve Masaüstü (≥1025px) için 3 farklı breakpoint tanımlandı.
- `Card` bileşeni eklendi.
